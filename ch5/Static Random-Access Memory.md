## 静态随机存储器哒

静态随机存储器以双稳态触发器为基本存储电路，存取速度快，保存的数据不需要刷新，但是集成度低，功耗大。
常用静态随机存储器（SRAM）芯片：
6116（2K×8）、2016、4016         
6264（8K×8）	
62128（16K×8）	
62256（32K×8） 	
62512（64K×8）
